# 크로노X 스크립트 변환기 설치 안내

## macOS 보안 경고 해결 방법

이 앱은 아직 Apple 개발자 인증서로 서명되지 않았기 때문에 처음 실행 시 보안 경고가 나타날 수 있습니다.

### 설치 방법

#### 방법 1: 우클릭으로 열기 (권장)
1. DMG 파일을 열고 앱을 Applications 폴더로 드래그
2. Applications 폴더에서 **크로노X 스크립트 변환기** 찾기
3. **Control 키를 누른 상태에서 클릭** (또는 마우스 우클릭)
4. 메뉴에서 **"열기"** 선택
5. 경고창에서 **"열기"** 버튼 클릭

#### 방법 2: 시스템 설정에서 허용
1. 앱을 처음 실행하여 오류 메시지 확인
2. **시스템 설정 > 개인정보 보호 및 보안** 열기
3. 하단에 "크로노X 스크립트 변환기가 차단되었습니다" 메시지 확인
4. **"확인 없이 열기"** 버튼 클릭

#### 방법 3: 터미널 사용 (고급 사용자)
```bash
sudo xattr -cr "/Applications/크로노X 스크립트 변환기.app"
```
